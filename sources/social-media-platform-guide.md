## Purpose

Reference document for social media content creation. Provides up-to-date platform specifications that prompts and skills use to ensure content fits each platform's constraints.

## Platform Specifications

### LinkedIn

| Parameter | Value |
|-----------|-------|
| Post character limit | 1,300 (organic), 700 (ads) |
| Article length | Up to 110,000 characters |
| Image sizes | 1200x627 (link share), 1080x1080 (single image) |
| Video length | 3 seconds to 10 minutes |
| Hashtag recommendation | 3-5 per post |
| Best posting times | Tuesday-Thursday, 8-10am and 12-1pm |
| Audience | Professionals, B2B, career-focused |

### Twitter/X

| Parameter | Value |
|-----------|-------|
| Tweet character limit | 280 |
| Thread limit | 25 tweets |
| Image sizes | 1600x900 (single), 700x800 (two images) |
| Video length | Up to 2 minutes 20 seconds |
| Hashtag recommendation | 1-2 per tweet |
| Best posting times | Monday-Friday, 9-11am |
| Audience | News-focused, real-time conversation |

### Instagram

| Parameter | Value |
|-----------|-------|
| Caption character limit | 2,200 |
| Bio character limit | 150 |
| Image sizes | 1080x1080 (square), 1080x1350 (portrait), 1080x566 (landscape) |
| Reel length | Up to 90 seconds |
| Story dimensions | 1080x1920 |
| Hashtag recommendation | 15-25 per post |
| Best posting times | Monday, Wednesday, Friday, 11am-1pm |
| Audience | Visual-first, lifestyle, younger demographic |

### Facebook

| Parameter | Value |
|-----------|-------|
| Post character limit | 63,206 |
| Image sizes | 1200x630 (link share), 1080x1080 (post) |
| Video length | Up to 240 minutes |
| Hashtag recommendation | 1-3 per post |
| Best posting times | Tuesday-Friday, 9am-12pm |
| Audience | Broad demographic, community-focused |
